Principais fomratações

Negrito / Destaque:

Nessa frase temos um termo em negrito usando a tag b.

Nessa frase temos um termo em destaque usando a tag strong.

Itálico / Ênfase:

Nessa frase temos um termo em itálico usando a tag i (não semântica).

Nessa frase temos um termo em ênfase usando a tag em (semântica).

Texto Marcado

Nessa frase temos um texto marcado usando a tag mark.

Podemos alterar o estilo do texto marcado com o style dentro do mark.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um texto como inserido para indicar que ele foi adicionado posteriormente.

Existe também o texto sublinhado utilizando a tag u (não semântica)

Texto Sobrescrito e Subescrito

Podemos marcar um texto como sobrescrito por exemplo x20 + 3 utilizando a tag sup.

Podemos marcar um texto como subescrito por exemplo H2O utilizando a tag sub.
